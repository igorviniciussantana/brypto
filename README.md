
# Brypto | CryptoWallet

A Brypto consiste em uma Progressive Web App capaz de gerenciar e explorar
criptomoedas baseada em uma API própria.

## Techs

- ReactJS
- NextJS
- Axios
- TypeScript
- next-pwa
